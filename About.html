About the Project

Rooster run is developed and maintained by 9 computer science students at Aston University. We came together to create a fun, interactive and multiplayer game which will be available to download by anyone soon on your desktop.

Come and play Rooster run now, and let's see if you have what it takes to escapes the farm and get your rooster to freedom and save it from becoming someone's sunday roast.

Rooster run will be available to our players (date). Create lobbies and share them with your friends to join and compete against each other.

We created this amazing game using libGDX which is a free open source game-development application framework. It allows for the development of desktop and mobile games by using JAVA programming language.

The Networking framework we chose to use was Kryonet, which is a Java library that provides a clean and simple API for efficient client/server network communication and especially good for games.
